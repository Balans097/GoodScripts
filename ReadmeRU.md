# 🐧 GNOME Nautilus Scripts: Автоматизация работы с файлами

> Набор bash-скриптов для контекстного меню Nautilus (GNOME Files), превращающих рутинные операции с документами и медиа в один клик.

## 🚀 Как это улучшает работу в GNOME/Linux?

GNOME придерживается философии минимализма: чистый интерфейс, отсутствие лишних окон, фокус на задачах. Однако встроенный файловый менеджер Nautilus намеренно не перегружен функциями конвертации или пакетной обработки. Эти скрипты закрывают этот пробел, сохраняя дух экосистемы:

- 🔹 **Интеграция без отрыва от контекста.** Всё доступно через стандартное контекстное меню (`ПКМ → Сценарии`). Не нужно открывать терминал, искать программу или перетаскивать файлы в тяжёлые GUI-конвертеры.
- 🔹 **Фоновая работа + нативные уведомления.** Скрипты не создают модальных окон. Обработка идёт в фоне, а результат или ошибка сообщаются через стандартную систему уведомлений GNOME Shell.
- 🔹 **Пакетность из коробки.** Выделяете папку или десятки файлов → запускаете скрипт → получаете готовый результат без ручного перебора.
- 🔹 **Полная приватность и контроль.** Все операции выполняются локально. Никаких облачных сервисов, телеметрии, водяных знаков или ограничений по размеру/количеству файлов.
- 🔹 **Прозрачность и лёгкость кастомизации.** Это обычный Bash + проверенные CLI-утилиты. Любой скрипт можно открыть, прочитать логику, изменить параметры или добавить поддержку новых форматов.
- 🔹 **Идеально для рабочих станций.** Экономит часы времени исследователям, авторам, разработчикам и всем, кто регулярно работает со сканами, документами, логами и медиафайлами.

## 📦 Что внутри?
Несколько примеров

| Скрипт | Описание | Ключевые зависимости |
|--------|----------|----------------------|
| `dav-файлы → mp4-файлы.sh` | Конвертация видео с видеорегистраторов (`.dav`) в MP4 без пережатия | `ffmpeg` |
| `djvu → pdf.sh` | Пакетное преобразование `.djvu`/`.djv` в PDF (поддержка файлов и папок) | `djvulibre` (`ddjvu`) |
| `Документы → PDF.sh` | Объединение и конвертация документов (Word, Excel, ODT, TXT, EPUB, HTML и др.) в единый PDF | `libreoffice`, `poppler-utils` |
| `Изображения → PDF.sh` | Создание PDF из изображений с сохранением оригинальных размеров | `img2pdf` |
| `Изображения → PDF (A4).sh` | То же самое, но с автоматическим масштабированием под формат A4 | `img2pdf` |
| `Объединить текстовые файлы в один файл.sh` | Рекурсивное слияние текстовых файлов/кода в один `.txt` с разделителями | `file` |
| `Разбить pdf-файл на фрагменты по 19 МБ.sh` | Разделение тяжёлых PDF на части ≤19 МБ (через `poppler-utils`) | `poppler-utils` |
| `Разбить pdf-файл на фрагменты по 19 МБ (qpdf).sh` | Более надёжная версия разбиения PDF через `qpdf` с верификацией реального размера | `qpdf` |
